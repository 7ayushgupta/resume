
## How to Use

### Requirements

A full TeX distribution is required. To install TeX Live:
```
sudo apt-get install texlive-full
```

### Usage

To compile the  [`cv.tex`](cv.tex) file, run:
```
xelatex cv.tex
```

This should result in the creation of [`cv.pdf`](cv.pdf)
